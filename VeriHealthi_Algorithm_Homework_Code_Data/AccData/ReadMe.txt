数据集介绍：
1. 数据集提供了 6轴 IMU 数据。
2. 数据格式：
    - 前5行为数据头，标注了采集数据的设备、采集数据的时间、采样率、数据精度、数据轴数。
    - 之后为IMU数据，7个为一组，依次为 X轴陀螺仪数据、Y轴陀螺仪数据、Z轴陀螺仪数据、X轴加速度计数据、Y轴加速度计数据、Z轴加速度计数据、调试数据（无需使用）。
3. 加速度计量程：-8G~+8G；数据分辨率： 4096/1G（一个重力加速度对应数值4096）。
4. 陀螺仪量程：-2000°/s~+2000°/s；数据分辨率：16.4/(1°/s）。
5. 数据集包含3类动作：走路（见walk子文件夹）、跑步（见run子文件夹）、其他（见others子文件夹）。并在数据集文件名中标注了每个文件的步数。
